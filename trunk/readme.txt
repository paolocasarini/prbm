PRBM 0.6.x

Requisiti
-jre 1.1.7 o successivi (funzionante)

Installazione:
- scompattare in una directory l'archivio contenente il programma
- aprire un prompt di comandi DOS
- andare nella directory nella quale si e' scompattato il programma
- digitare: java -jar prbm.jar
- premere invio
- have fun

Note: Se il JRE e' installato a modo per lanciare il programma e' sufficiente
fare doppio-click sulla icona con il nome prb.jar .
