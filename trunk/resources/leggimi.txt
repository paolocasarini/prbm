
      PRBM - Percorso Rettificato belga Multimediale
      ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

                     versione 0.6.9
   
Requisiti
~~~~~~~~~
-jre 1.5.0 o successivi (funzionante)

Installazione
~~~~~~~~~~~~~
1. scompattare in una directory l'archivio contenente il programma
2. aprire un prompt di comandi DOS
3. andare nella directory nella quale si e' scompattato il programma
4. digitare: java -jar prbm.jar
5. premere Invio
.. have fun

Note
~~~~
1. Se il JRE e' installato a modo per lanciare il programma e' sufficiente
   fare doppio-click sulla icona con il nome prb.jar .
